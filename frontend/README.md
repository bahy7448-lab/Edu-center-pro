# EduCenter Pro — Frontend

## طريقة التشغيل على VS Code

### 1. نزّل Node.js أولاً
https://nodejs.org (اختار LTS)

### 2. افتح المجلد في VS Code
افتح مجلد `frontend` مباشرة في VS Code

### 3. شغّل في الـ Terminal
```bash
npm install
npm run dev
```

### 4. افتح المتصفح
http://localhost:5173
